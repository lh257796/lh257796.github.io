Anchors (i.e. links)

anchor tags: link to other pages. URL in 'href'. Go to DuckDuckGo for more info

Setting 'target="_blank"' will open link in a new window. See MDN for more details.

Anchor tags can also create links to a new location on the same page. Instead of putting URL in the "href" attribute, just use the "id" of the destination element. Jump to top of the page
